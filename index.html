❓ Help & Tips ▼

🛡️ Data Safety
• Browser cache clear hone se data ja sakta hai
• Daily JSON backup le — "💾 Save" button se
• Profiles use karo — multiple offices ke liye
• Tool HTML file ko Drive/Email pe save kar lo
📦 Bulk Import workflow:
1. Excel banao with columns: NAME, FATHER'S NAME, DOB, EMPLOYEE CODE, BLOOD GROUP, CONTACT NUMBER, OFFICE ADRESS
2. Photos rename to EmpCode.jpg
3. Photos folder ZIP banao
4. Upload + click Import button
✈️ Offline use:
• Manual entry, edit, print — sab offline chalega
• Bulk Import ke liye internet chahiye (pehli baar)
• Saved JSON files anytime load ho sakti hain
🖨️ Print tips:
• Browser print dialog me "Actual Size" ya "100%" select karo
• "Fit to page" OFF rakho
• Card stock paper use karo (~250 GSM)
